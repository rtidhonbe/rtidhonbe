# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in rti dhonbe, please report it responsibly by emailing **rtidhonbe@proton.me**.

Do not open a public issue for security vulnerabilities.

## What to expect

- You will receive an acknowledgement within 48 hours
- We will investigate and provide updates as we work on a fix
- Once resolved, we will credit you in the fix (unless you prefer to remain anonymous)

## Scope

This policy covers the rti dhonbe codebase and the live deployment at rtidhonbe.com. It does not cover the ICOM Mahoali portal or any third-party services.
